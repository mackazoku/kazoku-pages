1. 소개

Kazoku는 귀하의 개인정보 및 개인 데이터 보호에 최선을 다하고 있습니다. 본 정책은 Kazoku 가족 관리 앱 사용 시 귀하의 정보를 어떻게 수집, 사용, 저장 및 보호하는지 설명합니다.

🛡️ 우리의 약속: 귀하의 가족 데이터는 가장 소중한 자산입니다. 우리는 귀하의 정보가 항상 안전하고 비공개로 유지되도록 최고 수준의 보안 조치를 적용합니다.

📊

2. 수집하는 정보

정보 유형 세부사항 사용 목적
개인 정보 닉네임, 프로필 사진/아바타, 생년월일 계정 생성 및 관리
가족 정보 구성원 닉네임, 생년월일 가족 조직 및 관리
활동 데이터 일정, 작업, 메모, 지출 관리 기능 제공
기기 정보 기기 유형, 운영 체제, 기기 ID 성능 최적화

⚠️ 참고: 서비스 제공에 필요한 정보만 수집합니다. 일부 데이터 수집을 거부할 수 있지만 사용자 경험에 영향을 줄 수 있습니다.

🎯

3. 정보 사용 방법

귀하의 데이터는 다음 용도로 사용됩니다:

앱 기능 제공 및 유지
기기 간 데이터 동기화
요청된 알림 및 리마인더 전송
성능 및 사용자 경험 개선
요청 시 기술 지원 제공
신규 기능 개발을 위한 사용 추세 분석

다음 용도로는 데이터를 사용하지 않습니다:

개인정보 판매 또는 임대
제3자 광고 또는 마케팅
앱 외부에서의 활동 추적
동의 없는 파트너와의 공유
🛡️

4. 데이터 보안

귀하의 데이터를 보호하기 위해 다층 보안을 적용합니다:

데이터 암호화:
저장 데이터의 AES-256 암호화
데이터 전송의 TLS 1.3 암호화
비밀번호의 SHA-256 해싱
접근 제어:
다단계 인증(2FA)
역할 기반 접근 제어
세션 시간 제한
보안 모니터링:
비정상 활동 감지
정기적인 데이터 백업
빈번한 보안 감사
🤝

5. 정보 공유

다음 경우에만 정보를 공유합니다:

가족 내: 귀하가 구성한 개인정보 설정에 따라
서비스 제공업체: 기밀 유지 계약 하의 기술 파트너
클라우드 저장 서비스(Google Cloud)
분석 서비스(Firebase Analytics)
푸시 알림 서비스(Firebase Cloud Messaging)
법적 요구사항: 당국의 요구 시

🔐 보증: 모든 파트너는 높은 보안 표준을 준수해야 하며 서비스 제공에 필요한 데이터에만 접근할 수 있습니다.

⚙️

6. 귀하의 권리

귀하는 개인 데이터를 완전히 제어할 수 있습니다:

접근: 당사가 저장한 모든 데이터 조회
편집: 개인정보 업데이트 또는 수정
데이터 삭제: 일부 또는 전체 데이터 삭제 요청
데이터 내보내기: 읽을 수 있는 형식으로 데이터 다운로드
수신 거부: 특정 데이터 처리 활동 제한
개인정보 설정: 정보를 볼 수 있는 사람 제어

📱 방법: 앱에서 공유 설정을 구성하세요.
