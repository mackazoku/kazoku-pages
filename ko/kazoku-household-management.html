🏠 가사 관리

모든 가족 구성원을 위한 가사 업무를 스마트하게 정리하고 할당하세요. 일상적인 집안일부터 정기적인 작업까지, 모든 것이 계획되고 추적되어 집이 깔끔하고 정돈된 상태를 유지합니다.

➕

작업 생성 및 할당

가사 작업 목록을 쉽게 생성하고 명확한 기한과 우선 순위로 각 구성원에게 할당합니다.

  • 카테고리, 우선 순위 및 담당자와 함께 새 작업 생성
  • 카테고리에서 작업 생성
  • 반복 작업을 위한 템플릿 생성
  • 저장된 템플릿에서 작업 생성
  • Kazoku AI 제안에서 작업 생성
  • 사용자는 각 작업에 대한 체크리스트를 생성할 수 있습니다
📊

작업 관리

모든 가족 구성원에게 친숙한 미니멀한 인터페이스로 가사 작업 목록을 관리합니다.

  • 가사 작업의 미니멀한 목록 표시
  • 담당자, 상태 및 업데이트 시간을 명확하게 표시
  • 작업에 대한 유용한 필터 제공
  • 작업 검색 기능
🏆

보상 포인트 시스템

작업을 잘 완료한 가족 구성원에게 포인트 및 보상 시스템을 통해 동기를 부여합니다.

  • 작업이 완료되면 자동으로 점수 부여
  • 어려운 작업에 대해 추가 포인트 허용
🔔

자동 알림

스마트 알림이 다가오는 작업이나 미완료 작업을 상기시켜 줍니다.

  • 모바일에서 푸시 알림
  • 마감일 전 알림
  • 빈도 및 시간 사용자 지정
  • 할당자를 위한 알림
🔗

다른 기능과의 통합

작업 관리는 다른 Kazoku 기능과 통합되어 포괄적인 가족 관리 도구를 제공합니다.

  • 가족 이벤트를 위한 작업 목록 관리
  • 여가 활동을 위한 작업 목록 관리
  • 활동과 연결된 작업도 통합 인터페이스에 표시

작업 관리 화면

작업 생성

작업 이름을 입력하고 카테고리, 날짜/시간, 위치를 선택하며 설명 및 이미지를 추가합니다. 작업을 분류하고 알림을 쉽게 설정할 수 있습니다.

작업 목록

모든 가족 구성원에게 친숙한 미니멀한 인터페이스로 가사 작업 목록을 관리합니다.

작업 세부정보

작업 세부 정보, 상태, 담당자, 설명 및 진행 상황을 확인합니다. 상태를 업데이트하고 빠르게 메모를 추가합니다.

연결된 작업 목록

이벤트 및 가족 활동과 관련된 작업을 표시합니다. 특정 상황에서 효과적인 작업 관리를 합니다.
